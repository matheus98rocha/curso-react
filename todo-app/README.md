## Installing / Getting started

```shell
npm init
npm i --save -E body-parser@1.15.2 express@4.14.0 mongoose@4.7.0 node-restful@0.2.5 pm2@2.1.5

npm i --save-dev -E nodemon@1.11.0

npm run dev
```
